Abstraction, as discussed in Chapter 11 of Expert One-On-One J2EE Design and Development by Rod Johnson (Wrox, 2002).
The classes in this package make JNDI easier to use, facilitate the accessing of configuration stored in JNDI and provide useful superclasses for EJB access classes.
In particular, application classes using this package do not need to catch naming exceptions or obtain JNDI contexts.
